Programming Languages

Programming Languages
Machine language An encoding system where instructions are coded as numeric digits
Debugging System of locating and correcting errors in the code
Assembler Converts mnemonic expressions into machine language instructions
Assembly language A mnemonic system for representing programs collectively
Translator Translates programs expressed in high-level primitives into machine-language programs
Compiler Compiles several machine instructions into short sequences to simulate the activity requested by a single high--level primitive
Interpreter Implement third-generation languages by actually executing a program from its high-level form rather than copying it and exectuing it later
Cross-platform software Software that is independent of an operating system's design as well as the machine's hardware design and is therefore executable throughout a network
Natural Language Languages shuch as English, German, or Latin that eveloved over time without formal grammatical analysis
Formal language Languages such as programming languages that are precisely define by grammars
Machine independence Where languages and programs could be compiled as easily for one machine as another and therefore be truly independent of machine
High-level programming language Provide more abstractions for the programmer and make it easier for people to read and write a program
Object-oriented programming (OOP) Software system viewed as a collection of units, called objects. Each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects. Together, these objects interact to solve the problem at hand.
Scripting language A subset of the imperative programming languages typically used to perform administrative tasks rather that to develop complex programs
Data types: Encompasses both the manner in which the data item is encoded and the operations that can be performed on that data
Integer Refers to numeric data consisting of whole numbers, probably stored using two's complement notation
Float Refers to numeric data that might contain values other than whole numbers, probably stored in floating-point notation
String Refers to alphabetical data that is entered
Primitive data type Data types that are included as primitives in a programming language, such as int for integer and char for character
Data structure This is the conceptual shape or arrangement of data
Python Collection Data Structures
List Used to store multiple items in a single variable
Tuple Also used to store multiple items in a single variable, but is ordered and unchangeable
Set Also used to store multiple items in a single variable, but is unordered, unchangeable, and unindexed
Dictionary Used to store data values in key: value pairs
Function Block of code that only runs when it is called
Local variable Container for storing data values
Global variable Variables that are created outside of a function
Source program Human readable computer instructions writeen by programmers
Object program A collection of data (variables) and methods (functions) that act on data
