Networking and Internet Terminology
TERM DEFINITION
Network Classifications
  • LAN - Local Area Network, usually consisting of a collection of computers in a single building or building complex
  • MAN - Metropolitan Area Network, is a network of intermediate size, such as one spanning a local community
  • WAN - Wide Area Network, links machines over a greater distance perhaps in neighboring cities or on opposite sides of the world
Network Models
  • Bus Network, computers are all connected to a common communication line called BUS
  • Star Network, one machine serves as a central focal point to which all the others are connected
Protocols
  • CSMA/CD Ethernet, Carrier Sense, Multiple Access with Collision Detection, dictates that each message be broadcast to all the machines on the bus.
  • CSMA/CA Wifi, Carrier Sense, Multiple Access with Collision Avoidance, wireless networks that adopt the policy of trying to avoid collisions.
Combining Networks
  • Repeater, a device that passes signals back and forth between the two original busses without considering the meaning of the signals.
  • Bridge, it connects two buses but only communicates messages when the destination is a computer on the other side.
  • Switch, essentially a bridge with multiple connections, allowing it to connect several buses rather than just two.
  • Router, connection between networks to form an internet
  • Gateway, the 'point' at which one network is linked to an internet
Client/Server Model
  • P2P: Peer-to-peer, involves processes that provide service to and receive service from each other.
  • Distributed System, consist of software units that execute as processes on different computers.
  • Cloud Computing, a distributed system in which many independent computers work closely together to provide computation or services comparable to a much larger machine.
Internet Addressing
  • Tier 1, very high speed, high-capacity, international WAns. Top of the hierarchy.
  • Tier 2, more regional in scope and less potent in their capabilities.
  • Tier 3, an access ISP is essentially an independent Internet, sometimes called an intranet providing access to individuals within their organizations.
IP Address
  • ICANN, Internet Corporation for Assigned Names and Numbers, non-profit corporation used to heirarchically assigned unique IP addresses- ICANN.org
  • DNS, Domain Name System, an Internet-wide directory system that provides address translation services to clients.
  • TLD: Top-Level-Domain, suffixes used to determine what sort of organization it is, like education, commercial institutions, or government etc...
  • Country-Code TLDs, two-letter suffixes that determine the Country-Code - Country-Codes
Internet Protocols
  • FTP, File Transfer Protocol, an application for accessing another computer from a great distance.
  • HTTP, Hyper Text Transfer Protocol, powerful webpage transfer protocol
  • SMTP, Simple Mail Transfer Protocol, a way that two computers on the network may interact when transmitting an email from one host to another.
World Wild Web
  • URL, Uniform Resource Locator, a unnique address containing the information needed by a browser to contact the proper server and request the desired document
  • Client-side Activities, activities performed by the client, such as filling out a form etc...
  • Server-side Activities, activities performed by the server
Internet Four-layer Software
  • Application Layer, consists of those software units such as clients and servers that use Internet communication to carry out their tasks.
  • Transport Layer, sends and receives packages
  • Network Layer, decides which direction a packet should be sent at each step along the packet's path through the Internet.
  • Link Layer, has the responsibility of transferring the packet.
Internet Is
  • Packet-switching Network, system through which digital data is sent by breaking the data into blocks of bits called packets.
  • Fault tolerant, if there is a breakdown in the transport system it will find another pathway.
  • Redundant, meaning that different routers are redundant if they breakdown another one can be used.
Cybersecurity
  • Malware, malicious software
  • Virus, software that infects a computer by inserting intself into programs that already reside in the machine.
  • Worm, autonomous program that thransfer itself through a network, taking up residence in computers and forwarding copties of itself to other computers.
  • trojan Horse, a program that enters a computer system disguised as a desirable program.
  • Spyware, collects information about activities at the computer on which is resides and reports that information back to the instigator of the attack.
  • Phishing, technique of obtaining information explicitly by simply asking for it.
  • DoS Attack, denial of service attack process of overloading a computer with messages.
Protection
  • Firewall, filters traffic passing through a point in the network.
  • Spam Filter, specific type of firewall used to block unwanted emails
  • Proxy Server, software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
  • Antivirus Software, used to detect and remove the presence of known viruses and other infections. Must be kept up to date.
  • Cryptography, study of sending and receiving messages securely in the presence of adversaries.
  • Public Key Encryption, involves the use of two keys. One key, known as the public key, is used to encrypt messages; the other key, known as the private key, is required to decrypt messages.
