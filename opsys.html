Operating Systems Definition Table

Operating Systems Terminology
TERM DEFINITION
operating system The software that controls the operation of the computer including file management, the user interface, and the program execution environment.
job queue The storage organization in which objects are ordered
FIFO First-in, first-out that is objects are removed from the queue in the order in which they arrived
interactive processing Allows a program being executed to carry on a dialogue with the user through remote terminals
time sharing An operating system that provides service to multiple users at the same time
multiprogramming A system where time is divided into intervals and then execution of each job is restricted to only one interval at a time
application software Programs for performing tasks particular to the machine's utilization
system software Performs those tasks that are common to computer systems in general
utility software Programs for performing activities that are fundamental to computer installations but not included in the operating system
graphical user interface (GUI) Objects to be manipulated, such as files and programs, are represented pictorially on the display as icons
shell Older user interfaces that communicated with users through textual messages using a keyboard and monitor screen.
kernel The internal part of an operating system
device driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine
file manager Job to coordinate the use of the machine's mass storage facilities
memory manager Coordinates the machine's use of main memory
virtual memory Through paging, how a computer shuffles extra pages to give the impression that there is more memory
boot strapping The process of transferring the operating system to the main memory when the computer is first turned on
ROM Read Only Memory is a construction of nonvolatile memory cells that are not erased when the computer is turned off. It's contents can be read but not altered
deadlock A situation when two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
spooling A technique of holding data for output at a later but more convenient time
super user or administrator Person with highly privileged access to the operating system identified during the login procedure
multi-factor authentication A process whereby more than one piece of evidence is required to gain access
